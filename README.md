## About

**Asignatura:** Estadistica · UAH GII · Curso 2024-25

Probability theory and statistical methods applied to engineering: data analysis, distribution fitting, hypothesis testing and regression.

## Topics covered

| Topic | Content |
|-------|---------|
| Descriptive statistics | Mean, variance, standard deviation, histograms, boxplots |
| Probability | Sample space, events, conditional probability, Bayes theorem |
| Distributions | Binomial, Poisson, Normal, Exponential, Student-t, Chi-squared |
| Estimation | Point estimators, confidence intervals, MLE |
| Hypothesis testing | Z-test, t-test, chi-squared test, p-value, Type I/II errors |
| Regression | Simple linear regression, correlation, R-squared |
| ANOVA | One-way and two-way analysis of variance |
